# Pomodoro Timer App

A simple and clean Pomodoro Timer built using **HTML, CSS, and JavaScript**.
This app helps improve productivity by using the Pomodoro technique: focused work sessions followed by short breaks.

## 🔗 Live Demo

https://seemantokd.github.io/Roadmap.sh-Project-Promodoro-App/

## 📌 Features

* 25-minute focus timer
* Short break timer
* Long break timer
* Start, pause, and reset functionality
* Active timer mode highlighting
* Clean and minimal user interface
* Fully responsive layout

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Timer logic and interactivity
* **Google Fonts** – Typography
* **Font Awesome** – Icons

## ⚙️ How It Works

The application follows the Pomodoro productivity technique:

1. Start a **Focus Session (25 minutes)**.
2. After the session ends, take a **Short Break (5 minutes)**.
3. After several sessions, take a **Long Break (15 minutes)**.

Users can switch between **Focus**, **Short Break**, and **Long Break** modes manually.

## 🚀 Getting Started

If you want to run the project locally:

1. Clone the repository

```
git clone https://github.com/seemantokd/Roadmap.sh-Project-Promodoro-App.git
```

2. Navigate to the project folder

```
cd Roadmap.sh-Project-Promodoro-App
```

3. Open `index.html` in your browser.
